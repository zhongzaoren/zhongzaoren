硬件部

实践是检验真理的唯一标准。

在学习理论的基础之上锻炼自己的实践能力，解决电脑的基本问题—计算机硬件的组装与维护，提升自我分析能力和动手能力，计算机系统的封装、安装、调试与维护，网络安全软件、系统辅助软件的使用等等硬件部都有所涉及。 同时，作为协会重要的组成部分，肩负着每月一次的大型活动—电脑义务维修的组织与举行。在这里，你可以亲手实践，给自己以锻炼机会的同时也帮他人解决了问题。更重要的是还能结识一群妹子哦~ 哦，还有男神


电子创新部

单片机 电子 电路

主要涉及基础的电子电路方面的应用，以及51单片机，Arduino等MCU的应用与开发。即使你从未接触过此类东西，只要你有兴趣，那么在这里，我们将带你探索电子的世界，在这里，你可以看到好多新奇的元件，芯片；可以自己动手，创造属于自己的玩具：奇妙的流水灯，变幻的光立方，表达心意的摇摇棒，在操作方面我们不仅可以采用C语言等编程语言，更是有图形化编程让成员快速上手，激发兴趣，在这里你可以尽情放飞自己的思维，用深邃的二进制体会电子世界的奥妙。


网站部

Html Css Js PHP等

图灵电脑协会网站部是专注于建设Web云应用的部门。网站部专注于HTML5 Web应用的开发，研究内容包括HTML、JavaScript、CSS、PHP、NodeJS等，覆盖了建设一个现代化网站所需要的大部分技能。通过学习与掌握以上技能，你可以构建、维护并美化你的博客，从0开始编写一个属于你自己的网站；还可以撰写网页时钟、2048等精巧的HTML5 小游戏。 我部一直以来都是齐鲁软件大赛参赛的中坚力量之一，部员曾经多次参加齐鲁软件大赛并获得不俗成绩。如果你也想要参加各种科创赛事、为自己赢得荣誉、锻炼编程能力，欢迎你加入我们部，我们部永远欢迎你！


网络安全部

拔小旗，拿渣站，你说咋办就咋办……

网络安全部专注于网络安全方面和少部分与网络安全相关的信息安全方面的研究，旨在通过网络攻防保护网络设备，软件的安全可靠，和保护数据不被恶意修改，在一次次实战中熟悉黑客入侵常用手法，对入侵做出响应策略 。小组合作，团队意识 为网络安全发展浴血奋战


秘书部

每天都是萌萌哒✧٩(ˊωˋ*)و✧

秘书部隶属十佳优秀社团图灵电脑社团。是整个社团最不要求技术也是最不可或缺的一个部门。秘书部存在目的：组织社团活动，宣扬社团文化，处理社团大小事务，记录社团人员资料。大大小小的活动要经过秘书部收拾打理然后设计计划开始实行。每周的教学任务由各部门自己组织，但是整个社团的面基聚会玩游戏还是要交给秘书部哒✧٩(ˊωˋ*)و✧秘书部虽然不像其他部门一样每周给社团成员讲知识，但是要清楚的记录每个成员的资料，要宣传社团文化，要记录社团发生的大小事。而且秘书部会随时准备为其他部门服务，为整个社团服务。
